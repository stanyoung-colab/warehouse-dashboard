# 仓库管理系统 - 数据看板

## 项目概述
基于物流仓库经营数据的可视化看板系统，用于展示库存、订单、成本等关键业务指标。

## 技术栈
- 前端：Vue 3 + Element Plus
- 后端：Node.js + Express
- 数据库：MySQL
- 部署：Docker + Nginx

## 项目结构
```
warehouse-dashboard/
├── backend/          # 后端服务
├── frontend/         # 前端应用
├── docker/           # Docker 配置
├── docs/             # 文档
└── deploy/           # 部署脚本
```
